# 🍔 Food Zone Website

A modern and responsive food restaurant website developed using HTML, CSS, and JavaScript. The website provides a clean and attractive interface to showcase delicious food items with a responsive layout.

---

## 📌 Project Overview

Food Zone is a front-end restaurant website created for learning and portfolio purposes. It features a modern homepage with a hero section, food gallery, navigation menu, and footer. The website is fully responsive and offers a simple user experience.

---

## ✨ Features

- 🍔 Modern Navigation Bar
- 🥗 Hero Section
- 🖼️ Food Gallery
- 📱 Responsive Design
